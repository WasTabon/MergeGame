# Iteration 1 — Foundation

## 1. Что было добавлено
Базовый каркас проекта: главное меню с переходом на игровую сцену, все core-менеджеры (звук, вибрация, переходы, валюта, попапы), пустой game-экран с HUD, сеткой кирок 5x4 и зоной блоков. Settings popup полностью рабочий.

## 2. Что изменилось с прошлой итерации
Первая итерация, изменений нет.

## 3. Editor скрипты — порядок запуска

После импорта файлов в Unity проект:

1. Открой меню `Tools / Merge Mining / (Iteration 1) Setup ALL Scenes` — создаст обе сцены и добавит их в Build Settings.

Альтернативно по отдельности:
- `Tools / Merge Mining / (Iteration 1) Setup MainMenu Scene`
- `Tools / Merge Mining / (Iteration 1) Setup Game Scene`

После выполнения сцены окажутся в `Assets/MergeMining/Scenes/`.

## 4. Как тестировать

1. Открой `Assets/MergeMining/Scenes/MainMenu.unity` и нажми Play
2. На экране меню — заголовок MERGE MINING, кнопка PLAY, шестерёнка Settings, сверху счётчики Coins (50) и Gems (5)
3. Нажми шестерёнку → откроется попап Settings с тогглами SFX / Music / Vibration, двумя слайдерами громкости и кнопкой RESET PROGRESS. Закрытие — кнопка CLOSE или тап по фону
4. Нажми PLAY → должен пройти fade в чёрное и обратно → загрузится Game scene
5. На Game scene — тот же HUD сверху, кнопка паузы справа, зона блоков (плейсхолдер с подсказкой), сетка 5x4 пустых слотов снизу, кнопка SHOP (пока неактивна)

## 5. Ожидаемый результат
- Все кнопки имеют scale-punch анимацию при тапе
- Шестерёнка → попап появляется через scale 0→1 с OutBack, исчезает с InBack, backdrop фейдит до 0.6 alpha
- Тоггли работают и сохраняют состояние между запусками
- Слайдеры работают, громкости сохраняются
- RESET PROGRESS очищает PlayerPrefs (после перезапуска сцены счётчики обнулятся)
- Переход MainMenu → Game через плавный fade
- HUD счётчики анимированно меняются если изменить значение (визуально пока не на чем проверить — будет в итерации 2)

## 6. Известные ограничения текущей итерации
- Кнопка SHOP неактивна (итерация 2)
- Кнопка PAUSE без логики (итерация 4)
- Кирок и блоков на сцене нет — итерации 2 и 3
- Звуковых файлов в архиве нет, SoundManager работает, но без клипов выдаёт warning в консоль
- Вибрация на iOS через Handheld.Vibrate работает грубо — на устройстве заметна, в редакторе игнорируется
