# Iteration 7 — Boosters

## 1. Что было добавлено
4 бустера: x2 Speed (кирки атакуют в 2 раза быстрее), x2 Gold (награды x2), Auto Merge (каждые 2 сек авто-мерж случайной пары), Instant Break (мгновенно уничтожает все блоки в ряду). Все бустеры за gems. Длительность 30 сек, Instant — единоразовое использование. UI-полоска бустеров внизу, прогресс-бар активного бустера сверху каждой кнопки, рамочное свечение по краям экрана пока активны Speed/Gold бустеры.

## 2. Что изменилось с прошлой итерации
- В Resources создан `BoosterConfig.asset` с 4 бустерами
- На Canvas добавлен `BoosterManager` — singleton, управляет таймерами и эффектами
- На Game scene появилась `BoosterHud` — 4 кнопки в горизонтальном ряду снизу
- Добавлен `BoosterEdgeGlow` — image на весь экран, светит когда активен Speed или Gold бустер (раз/два секунды пульс)
- `PickaxeMiningBehaviour` теперь учитывает `BoosterManager.SpeedMultiplier`
- `BlocksRowManager` теперь учитывает `BoosterManager.RewardsMultiplier` при coin burst
- `PickaxeGridManager` получил метод `TryAutoMergeOneRandom` для Auto Merge бустера
- Сетка кирок и кнопка SHOP сдвинуты немного вверх, чтобы освободить место для бустеров внизу

## 3. Editor скрипты

`Tools / Merge Mining / (Iteration 7) Update Game Scene`

## 4. Как тестировать

### Старт:
1. Открой Game scene → внизу должна появиться полоска из 4 кнопок бустеров
2. Над каждой кнопкой — название, цена в gems, выше — цветной значок
3. Кнопки активны если у тебя достаточно gems (10/10/15/5 соответственно)

### x2 Speed:
4. Тапни x2 SPEED (стоит 10 gems) → -10 gems, кнопка теряет cost-row, появляется обратный отсчёт "30s" сверху
5. Внутри кнопки появляется заливка от верху вниз (cooldown overlay) — заполнение уменьшается с 1 до 0 за 30 сек
6. По краям экрана появляется голубое пульсирующее свечение
7. Кирки начинают атаковать в 2 раза чаще — заметно глазом
8. Через 30 сек бустер кончается, свечение пропадает

### x2 Gold:
9. Тапни x2 GOLD → жёлтое свечение, кирки атакуют как обычно, НО разрушенный блок даёт в 2 раза больше coins
10. Можно совмещать со Speed — оба активны одновременно

### Auto Merge:
11. Тапни AUTO MERGE (15 gems) → начнёт каждые 2 сек авто-мержить случайную доступную пару на сетке. Если на сетке нет пар одинакового уровня — ничего не делает
12. Через 30 сек выключается

### Instant Break:
13. Тапни INSTANT (5 gems) → ВСЕ текущие блоки в ряду мгновенно разрушаются → explosion'ы + coin burst х4 (один на каждый блок)
14. Через 0.35s появляется новый ряд блоков. Очень эффективно когда блоки в Lava Core с гигантским HP, и нужно быстро прокачать прогресс

### Если нет gems:
15. Кнопка становится серой/полупрозрачной — не нажать. Накопить gems можно через Zone Complete и сундуки

## 5. Ожидаемый результат
- При активации звук/вибрация → бустер реально работает
- Cooldown overlay видно как падающая заливка
- Speed и Gold могут быть одновременно (двойной buff)
- Auto Merge даёт удобство, не ломая контроль игрока (он всё равно может тащить кирки руками)
- Instant Break даёт "сейф" если игрок попал в туго работающую зону

## 6. Известные ограничения
- Бустеры не сохраняются между сессиями — если перезапустить во время активного бустера, он пропадёт (это окей для прототипа, но финальная игра должна сохранять оставшееся время)
- Auto Merge выбирает случайную пару, не оптимальную (например 1+1 может мержиться раньше чем 4+4)
- Cooldown overlay показывает оставшееся время как vertical-fill, без числа секунд внутри — но число "30s" есть текстом сверху
