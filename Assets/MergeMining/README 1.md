# Iteration 3 — Blocks + Mining Loop

## 1. Что было добавлено
Блоки (Stone / Iron / Gold) с HP-баром в ряд сверху, кирки автоматически выпускают атаки-снаряды по блокам, при разрушении блока — explosion + 6 монеток летят к HUD и пополняют счётчик coins. Тап по блоку — +1 урона.

## 2. Что изменилось с прошлой итерации
- В `BlocksRow` убран плейсхолдер-текст, появились 4 `BlockSlot_X` с реальными блоками
- На `PickaxePrefabTemplate` добавлен `PickaxeMiningBehaviour` — все новые и существующие кирки теперь автоматически атакуют
- В `PickaxeGridManager` появилось поле `miningAttackTemplate` (заполняется автоматически editor скриптом)
- На сцене появились скрытые шаблоны: `BlockPrefabTemplate`, `MiningAttackTemplate`, `BlockDestroyEffectTemplate`, `CoinBurstParticleTemplate`
- В Resources создан `BlockConfig.asset` (4 типа: Stone, Iron, Gold, Crystal — последний пока в последовательности не используется, но готов для иттер 5)

## 3. Editor скрипты — порядок запуска

В Unity:
1. `Tools / Merge Mining / (Iteration 3) Update Game Scene`

Скрипт сам создаст BlockConfig.asset, добавит блоки и менеджеры, обновит сцену.

## 4. Как тестировать

1. Запусти MainMenu, жми Play, перейди в Game scene
2. Сверху должны быть 4 блока:
   - 2 серых Stone, 1 бронзовый Iron, ещё 1 Stone (порядок по последовательности из BlockConfig)
   - У каждого есть HP-бар сверху (зелёный → жёлтый → красный по мере уменьшения)
   - Под блоком надпись "STONE" / "IRON" / "GOLD"
3. Снизу — 2 кирки lvl 1, они автоматически каждую секунду выпускают летящие "снаряды" по случайному блоку
4. Снаряд летит по дуге, при попадании блок дёргается, HP-бар уменьшается, мини screen shake
5. Когда HP=0 → explosion (shockwave + 12 частиц + 6 "кусочков" блока разлетаются с вращением), screen shake побольше, haptic Medium
6. 6 монеток вылетают из позиции блока, разлетаются веером, затем летят к иконке coins в HUD, при достижении +N coins
7. Через 0.35s появляется новый блок (тип следующий по последовательности)
8. Тап по блоку → +1 урона, scale punch, мини haptic Light
9. По мере роста уровня кирок (через merge) — урон растёт, мини-кирки летят чаще
10. Цена SHOP при покупке отрабатывает деньги — теперь можно заработать coins, нет нужды ресетить прогресс

## 5. Ожидаемый результат
- Loop замыкается: кирки добывают → coins прилетают → можно купить кирок → мержить → ещё быстрее добывать
- HP блоков растёт по формуле `10 × 1.18^N` (где N — всего разрушено за всю игру), reward `3 × 1.15^N`
- Iron имеет ×1.4 HP и ×1.6 reward от формулы
- Gold ×2 HP и ×3 reward
- При перезапуске сцены: блоки спавнятся свежие, но HP уже учитывает накопленный total_blocks_destroyed (т.е. прогресс не сбрасывается)
