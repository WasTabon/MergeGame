# Iteration 6 — Chests + Premium Currency

## 1. Что было добавлено
3 типа сундуков (Free / Block / Premium), таймер Free Chest (каждые 5 минут), Block Chest (каждые 20 разрушенных блоков), Premium Chest за 50 gems. Полноценный chest opening попап с анимацией тряски, открытия крышки, вспышки и появления наград. Сидячая ChestHud-панель между блоками и сеткой кирок.

## 2. Что изменилось с прошлой итерации
- В Resources создан `ChestConfig.asset` с 3 сундуками и пулом наград (coins, gems, pickaxe lvl1-6)
- На Game scene появилась `ChestHud` — горизонтальная панель из 3 кнопок сундуков
- На Game scene появился `ChestOpeningPopupCanvas` (sortingOrder=215, pausesGame=true)
- На Canvas добавлен `ChestManager` — singleton, управляет состоянием сундуков, рандомом наград
- `BlocksRowManager.OnBlockDestroyed` теперь зовёт `ChestManager.RegisterBlockDestroyed()` — счётчик блоков для Block Chest

## 3. Editor скрипты

`Tools / Merge Mining / (Iteration 6) Update Game Scene`

## 4. Как тестировать

### Free Chest (с таймером):
1. Открой Game scene — между блоками и сеткой кирок видна полоска из 3 сундуков
2. Free Chest готов на старте (первый запуск, last time = 0) — кнопка активна, надпись "READY", красный значок-точка справа сверху
3. Тапни — попап ChestOpeningPopup: chest появляется с пружинкой, дрожит, крышка откидывается + flash, 4 reward'а раскрываются с задержкой, кнопка CONTINUE
4. Награды добавляются в инвентарь: coins/gems в счётчик HUD, кирки спавнятся в свободные слоты сетки
5. CONTINUE — закрытие, таймер Free Chest начинает идти от 5:00 вниз

### Block Chest:
6. Под Block-сундуком "X/20" — счётчик разрушенных блоков от последнего сундука
7. Разрушай блоки — счётчик растёт
8. На 20-м блоке — "READY" + значок — попап с 5 наградами (более ценные чем Free)

### Premium Chest:
9. Стоимость 50 gems. Кнопка неактивна если меньше 50 gems
10. С Zone Complete (даёт +1..+10 gems) накопи 50, открой Premium — попап с 6 наградами (включая high-level pickaxes)

### Корректность сохранения:
11. Тапни Free Chest, перезапусти сцену — таймер продолжает отсчёт (не сброситься на 5:00)
12. На 15/20 разрушено, перезапусти — счётчик остаётся 15/20
13. После открытия Block Chest — счётчик обнуляется

### Если кирок-наград некуда сложить:
14. Заполни сетку 20 кирками, открой сундук — кирки конвертируются в coins (level × 25) автоматически

## 5. Ожидаемый результат
- Все 3 сундука работают, награды реально начисляются
- Таймер Free Chest идёт в реальном времени (даже если закрыл/открыл приложение — таймер посчитает разницу)
- Анимация открытия "сочная" — pop появление, тряска, открытие крышки, flash, поэтапное появление наград

## 6. Известные ограничения
- Реклама-за-открытие-Free нет
- Звуковые ассеты — placeholder через SoundManager warnings
- Visual сундука простой — два прямоугольника body+lid + lock-кружок
